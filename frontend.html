Front-End

Frontend es la parte de un programa, sitio web o dispositivo a la que un usuario puede acceder directamente. En el contexto de diseño web y desarrollo web, se refiere a todas las tecnologías que corren en el navegador y que se encargan de la interactividad con los usuarios.


La programación es otro aspecto clave del desarrollo front-end. Utilizando lenguajes como HTML, CSS y JavaScript, se crea la estructura, el estilo y la interactividad de las páginas web. Dominar estos lenguajes y sus conceptos básicos es fundamental para crear y mantener sitios web funcionales y atractivos.
